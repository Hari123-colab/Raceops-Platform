# Raceops-Platform
Cloud-native motorsport telemetry and analytics platform using Python, FastAPI, PostgreSQL, AWS, Terraform, Docker, Kubernetes, Prometheus and Grafana.
Motorsport Telemetry & Analytics Platform

### Overview

A cloud-native motorsport telemetry and analytics platform designed to simulate race-car sensor data, process telemetry events, perform performance analytics, and visualize race metrics.

## Objectives

- Simulate motorsport telemetry
- Build analytics pipelines
- Store telemetry data
- Visualize race performance
- Deploy on AWS
- Provision infrastructure using Terraform
- Run workloads on Kubernetes

## Technology Stack

- Python
- FastAPI
- PostgreSQL
- Grafana
- Docker
- AWS
- Terraform
- Kubernetes
